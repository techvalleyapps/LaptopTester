📥 Full Battery Test App
Download the Windows diagnostic app — it auto-reads design capacity, full charge capacity, calculates health, and generates the official Windows battery report. No copy-pasting needed.
🟦 Option A — Easy (Recommended)
Double-click the .bat file. It automatically requests Administrator rights and launches the tester.
⬇ RunBatteryTest.bat
  1. Download the .bat file
  2. Double-click it
  3. Click Yes on the UAC prompt
⚙️ Option B — PowerShell Script
If you prefer to run PowerShell directly or inspect the script first.
⬇ BatteryTest.ps1
  1. Download the .ps1 file
  2. Right-click → Run with PowerShell
  3. Allow Administrator if prompted

The app shows: charge level · battery status · design capacity · full charge capacity · health % · and generates a full powercfg /batteryreport with one keypress.

💻 Quick Commands
Copy and run in PowerShell or Command Prompt as Administrator.
Full battery report (saves HTML to your profile):
powercfg /batteryreport /output "%USERPROFILE%\battery-report.html"
Energy efficiency scan (~60 s):
powercfg /energy
Design capacity via PowerShell:
(Get-WmiObject -Class BatteryStaticData -Namespace root\wmi).DesignedCapacity
Full charge capacity via PowerShell:
(Get-WmiObject -Class BatteryFullChargedCapacity -Namespace root\wmi).FullChargedCapacity
